语句

  • 程序由语句构成
  • 所有的语句必须以分号结尾
  • 例如：console.log('hello');
  • 这个是合法的语句
  • console.log('hello')
  • 这个是有问题的。
  • console.log('hello');console.log('hello');
  • 这是两个语句， 虽然我们只写了一行
  • 推荐的写法是：
  • console.log('hello');
  • console.log('hello');

以下哪个语句是正确的？

  • var a = 3+4;
  • var a = 3+'4'
  • var a = 3+4.
  • var a = 3+4

正确答案

  • 答案是第一个。 其他几个都不是分号结尾的， 因此不是语句

    • 下面哪个选项有两个语句构成？

    • var a = 3+4;var b = 3+5
    • var a = 3+4;var b = 3+5;
    • var a = 3+4;var b = 3+5;var c = 3+6;
    • var a = 3+4;var b = 3+5;var c = 3+6

    正确答案

    • var a = 3+4;var b = 3+5
    • 分号前面是一个语句， 后面的因为不是分号结尾， 因此不是语句
    • var a = 3+4;var b = 3+5;
    • 有两个语句， 这个是正确答案
    • var a = 3+4;var b = 3+5;var c = 3+6;
    • 这个有三个语句
    • var a = 3+4;var b = 3+5;var c = 3+6
    • 这个有两个合法语句， 后面的不是语句。 因此整段程序不可用。 不可用的程序谈不上有几个语句

    定义变量

    • 变量：用字母来表示一个数值
    • 与数学的方程不同的是， 变量可以由多个字母([a-z], _, $)以及数字构成()
    • 注意数字不能做开头, 不能有空格， 因为空格是分隔符。
    • 例如：aaa是合法变量
    • aaa3是合法变量
    • 3aaa是非法变量， 因为数字不允许做开头。

    以下哪个变量是合法的？

    • 3Cat
    • Cat 3
    • mouse_4
    • $person

    答案

    • 3Cat 非法， 数字开头
    • Cat 3 非法， 有空格
    • _mouse_4 合法， 使用下划线做开头
    • $person 合法， 使用$做开头

    定义变量与赋值

    • 定义变量， 使用var关键字。
    • 例如： var a;
    • 这样就定义了一个变量， 叫a， 后面就可以使用a这个变量了
    • 这个时候， 这个变量是没有值的
    • 定义完变量， 变量就可以被赋值了。
    • 赋值使用等号。 例如：a=3;
    • 这个时候a就有值了， 值为3.
    • 如果再做一次赋值， a=4;
    • 那么a就为4了， 不再等于3

    定义的同时赋值

    • 通常， 我们在定义变量的同时， 会给一个初始值。 我们可以在定义的同时做赋值
    • 例如：
    • var a = 3;
    • 这样我们就定义了一个变量叫a， 并且a的值为3
